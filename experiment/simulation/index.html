Simulator for Square Wave Generation using 8051 Microcontroller

Input

Delay:

Code

Squarewave: CPL P1.0

ACALL 50

SJMP Squarewave

Instructions

  1. First calculate the delay time using frequency and write it in the Delay input box.
  2. Then choose the input as 0 or 1 by clicking on the corresponding buttons.
  3. In the first statement we complement the contents of port 1.0.
  4. Then we call the delay program and the square wave is generated.
  5. Then we jump back to the first statement.
  6. The student can pause the waves and calculate the frequency by observing the time period of the waves and verify.
